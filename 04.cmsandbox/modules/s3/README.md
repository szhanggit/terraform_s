# AWS S3 Buckets

This module creates the necessary S3 buckets for the application.

## Variables

- `environment`: The environment for which the S3 bucket is being created (e.g., dev, staging, prod).
- `cerbos_policies_bucket_name`: The name of the S3 bucket to create for storing Cerbos policies.
- `cerbos_config_bucket_name`: The name of the S3 bucket to create for storing Cerbos configuration.
